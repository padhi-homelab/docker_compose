### Supported Architectures

- ~~`x86` (`linux/386`)~~
- `amd64` (`linux/amd64`)
- `arm` (`linux/arm/v6`)
- `armhf` (`linux/arm/v7`)
- `aarch64` (`linux/arm64`)
